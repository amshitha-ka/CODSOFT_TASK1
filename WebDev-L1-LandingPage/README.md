# Level 1 - Task 1: Landing Page

## Overview
A modern, responsive landing page created as part of the **Oasis Infobyte Internship Program (OIBSIP)** for the **Web Development & Designing** track.

## Features
- **Header & Navigation:** Fixed navigation bar with direct jump links and call-to-action button.
- **Hero Banner:** Eye-catching headline with primary and secondary CTA buttons.
- **Features Section:** Clean 3-column feature showcase grid built with CSS Grid.
- **About Section:** Clear concise summary of company/project goals.
- **Contact Form:** Interactive form with live client-side validation.

## Built With
- **HTML5**
- **CSS3** (Flexbox, CSS Grid, Responsive Design)
- **JavaScript** (DOM Manipulation & Form Validation)
