# Level 1 - Task 3: Temperature Converter Website

## Overview
A real-time temperature conversion web application built for the **Oasis Infobyte Internship Program (OIBSIP)** under the **Web Development & Designing** track.

## Features
- Input validation to handle numeric and invalid inputs gracefully.
- Converts dynamically between **Celsius**, **Fahrenheit**, and **Kelvin**.
- Displays clean output rounded to two decimal places.
- Clean, responsive UI built with modern CSS flexbox styling.

## Built With
- **HTML5**
- **CSS3**
- **JavaScript** (DOM Events & Mathematical Conversions)
